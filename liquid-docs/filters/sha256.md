# sha256 Filter Documentation

## Overview
The `sha256` filter transforms input strings into SHA-256 hashes within Liquid templates.

## Syntax
```
string | sha256: string
```

## Return Type
Returns a [string](https://shopify.dev/docs/api/liquid/basics#string)

## Description
"Converts a string into an SHA-256 hash." However, the documentation recommends using the blake3 filter instead, citing "better security and performance."

## Example

**Input Code:**
```liquid
{%- assign secret_potion = 'Polyjuice' | sha256 -%}
My secret potion: {{ secret_potion }}
```

**Output:**
```html
My secret potion: 44ac1d7a2936e30a5de07082fd65d6fe9b1fb658a1a98bfe65bc5959beac5dd0
```

## Deprecation Notice
This filter is discouraged in favor of the blake3 filter for enhanced cryptographic security and improved performance characteristics.
